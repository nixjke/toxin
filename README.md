# Toxin

## Hotel website to search for rooms and rent

[View Toxin](https://nixjke.github.io/toxin/)

## Installation

```bash
git clone https://github.com/SosiskaPish/toxin.git
cd toxin
npm install
```

### Local dev server

| | Version |
|---|---|
| node | 14.16.0 |
| npm | 6.14.11 |

```bash
npm start
```

### Build Setup

```bash
npm run build
```

### Deploy gh-pages

```bash
npm run deploy
```
