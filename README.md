# Toxin

## Installation

```bash
git clone https://github.com/SosiskaPish/toxin.git
cd toxin
npm install
```

### Local dev server

```bash
npm run dev
```

### Build Setup

```bash
npm run build
```

### Deploy gh-pages

```bash
npm run deploy
```
